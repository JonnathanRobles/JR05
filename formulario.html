Crea una página web que contenga un formulario con los siguientes campos de información:

Título

Nombres, con un control de tipo texto obligatorio y con autofoco.

Email, con un control de tipo email y obligatorio.

Web, con un control de tipo url que muestre la ayuda "URL de tu página web personal".

Fecha de Registro, con un control de tipo date.

Hora, con un control de tipo time.

Edad, con un control de tipo number que limite la entrada a un valor entre 18 y 65.

Contacto, con un control de tipo tel.

Color favorito, con un control de tipo color.

Un botón de envío.

Considera lo siguiente

El título de la página debe ser Ejercicio Formulario HTML.

El método de envío del formulario debe ser GET.

El destino del envío del formulario debe ser "https://www.utpl.edu.ec/".
